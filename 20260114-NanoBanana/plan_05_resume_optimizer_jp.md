# ビジネスプラン 05: AI履歴書＆LinkedIn最適化ツール (AI Resume & LinkedIn Optimizer)

## 1. 概要
求職者の履歴書（レジュメ）やLinkedInプロフィールをAIが分析し、希望する職種のキーワードに合わせてリライト・最適化するツール。採用担当者の目に留まりやすい形に自動修正する。

## 2. ターゲット顧客
- 就職・転職活動中の人
- キャリアアップを目指すプロフェッショナル
- 自分の経歴をうまく言語化できない人
- 英語での履歴書作成に自信がない非ネイティブスピーカー

## 3. 解決する課題
- **書類選考落ち**: 能力はあるのに、履歴書の書き方が悪くてATS（採用管理システム）で弾かれてしまう。
- **アピール不足**: 自分の強みを客観的に表現するのが難しい。
- **時間の浪費**: 応募する企業ごとに履歴書を書き直すのが大変。

## 4. 提供価値
- **通過率アップ**: ATS対策（キーワード最適化）により、採用担当者の目に触れる確率を高める。
- **プロフェッショナルな文章**: 稚拙な表現を、ビジネスに適したパワフルな表現（Action Verbsの使用など）に変換。
- **一貫性**: 履歴書の内容とLinkedInのプロフィールを整合させ、信頼性を高める。

## 5. 具体的な実装方法・ツール
- **主要ツール**: OpenAI API, LinkedIn API (情報の取得が可能であれば), PDF解析ライブラリ
- **フロー**:
    1.  現在の履歴書（PDF/Word）をアップロード、またはLinkedInのURLを入力。
    2.  希望する職種（Job Description）を入力。
    3.  AIがギャップ分析を行い、強調すべきスキルや経験を提案。
    4.  最適化された履歴書と、LinkedInの「About」セクションの文章を生成。

## 6. 収益化モデル
- **フリーミアム**: 基本診断は無料、リライトとダウンロードは有料（1回 $5 または 月額 $15）。
- **プレミアムサービス**: 生成された履歴書に対する人間（キャリアコーチ）による最終チェックオプション。
