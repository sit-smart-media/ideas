# ビジネスプラン 06: AIパーソナライズ学習プランナー (AI Personalized Study Planner)

## 1. 概要
学生や資格試験の受験生向けに、学習習慣、目標、試験日、得意・不得意科目を分析し、最適な日々の学習スケジュールを自動生成するツール。「賢く学ぶ」を支援する。

## 2. ターゲット顧客
- 大学受験生
- 資格試験（TOEIC, 簿記など）の受験者
- 社会人学習者
- 教育機関・塾

## 3. 解決する課題
- **計画倒れ**: 無理なスケジュールを立ててしまい、すぐに挫折する。
- **効率の悪さ**: 既に理解している部分に時間をかけ、苦手な部分がおろそかになる。
- **モチベーション維持**: 何をどれだけやれば合格できるかが見えず、不安になる。

## 4. 提供価値
- **最適化**: 忘却曲線などを考慮し、最も定着しやすいタイミングで復習をスケジュールに組み込む。
- **柔軟性**: 「今日は体調が悪いので休んだ」などの突発的な事態に合わせて、計画を自動で再調整（リスケジュール）。
- **見える化**: 進捗状況を可視化し、達成感を与えることで継続を促す。

## 5. 具体的な実装方法・ツール
- **主要ツール**: OpenAI API, React Native/Flutter (モバイルアプリ), Google Calendar API
- **フロー**:
    1.  試験日と目標スコア、1日の学習可能時間を入力。
    2.  現状のレベルチェックテスト（または自己申告）を行う。
    3.  AIが試験日から逆算して、日ごとの詳細なタスクリストを作成。
    4.  学習完了ボタンを押すと進捗が記録され、遅れが出れば翌日以降のプランを自動修正。

## 6. 収益化モデル
- **サブスクリプション**: 広告なし、無制限のリスケジュール機能（月額 $5〜$10）。
- **B2B販売**: 塾や予備校の生徒管理システムの一部としてライセンス提供。
