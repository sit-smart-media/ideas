# ビジネスプラン 13: AIマイクロ言語チューター (AI Tiny Language Tutor)

## 1. 概要
言語学習者に対し、その人のレベルや興味に合わせた「マイクロレッスン（数分で終わる学習）」と、リアルタイムの会話練習・訂正を提供するAIチューター。

## 2. ターゲット顧客
- 忙しい社会人（隙間時間で勉強したい）
- 語学学校に通う予算や時間がない人
- 実践的な会話力をつけたい人
- 留学準備中の学生

## 3. 解決する課題
- **継続の難しさ**: まとまった勉強時間が取れず、挫折してしまう。
- **アウトプット不足**: 単語は覚えても、実際に使う機会がない。
- **羞恥心**: 人間の先生相手だと間違えるのが恥ずかしくて話せない。

## 4. 提供価値
- **手軽さ**: LINEやWhatsAppなどの使い慣れたアプリ上で、1日5分から学習可能。
- **即時フィードバック**: 発言した直後に、文法ミスやより自然な表現をAIが優しく指摘。
- **興味関心ベース**: 「好きな映画」や「仕事の内容」など、ユーザーが興味のある話題で会話できる。

## 5. 具体的な実装方法・ツール
- **主要ツール**: OpenAI API (GPT-4), Whisper (音声認識), TTS (音声合成)
- **フロー**:
    1.  その日のトピック（例：「カフェでの注文」）が通知される。
    2.  ユーザーがボイスメッセージまたはテキストで回答。
    3.  AIが返信し、会話を広げる。同時に、文法や発音のポイントをフィードバック。
    4.  単語帳機能に新しい単語を自動登録。

## 6. 収益化モデル
- **サブスクリプション**: 1日あたりの会話数制限解除、詳細な学習レポート（月額 $5〜$15）。
- **法人契約**: 社員の語学研修としての導入。
