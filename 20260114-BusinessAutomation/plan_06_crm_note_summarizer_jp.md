# ビジネスプラン 06: CRMノート要約・入力自動化 (CRM Note Summarizer)

## 1. 概要
商談や会議の録音データを自動で文字起こしし、重要なポイント、顧客の要望、ネクストアクションを抽出してCRM（顧客管理システム）に自動入力するソリューション。

## 2. ターゲット顧客
- フィールドセールス・インサイドセールスチーム
- カスタマーサクセス部門
- 法律事務所・コンサルティングファーム（記録が重要な業種）

## 3. 解決する課題
- **入力漏れ・忘れ**: 忙しい営業担当が商談後の入力を後回しにし、情報が欠落する。
- **情報のブラックボックス化**: 担当者しか詳細を知らず、チームで共有できない。
- **事務作業の負荷**: 1時間の商談のまとめに20分かかるといった非効率（週2〜4時間のロス）。

## 4. 提供価値
- **完全な記録**: 自動化により、重要な発言の記録漏れがなくなる。
- **即時共有**: 会議終了後すぐにチーム全体で情報を参照可能。
- **リソース最適化**: 営業担当は「入力作業」ではなく「顧客との対話」に集中できる。

## 5. 具体的な実装方法・ツール
- **主要ツール**: Otter.ai, Fireflies.ai, OpenAI API, Salesforce/HubSpot
- **フロー**:
    1.  Web会議ツール（Zoom/Meet）に自動記録ツールを連携。
    2.  会議終了後、トランスクリプト（文字起こし）を生成。
    3.  AIが要約と重要項目（予算、決裁者、課題など）を抽出。
    4.  CRMの該当フィールドにデータを自動流し込み。

## 6. 収益化モデル (ビジネスとしての展開)
- **SFA/CRM導入・改善支援**: Salesforce等の導入支援の一部として、この自動入力フローを構築（プロジェクト単価 $3,000〜）。
- **セールスイネーブルメント研修**: 営業チーム向けに、AIツールを使った効率的な商談記録メソッドを指導。
